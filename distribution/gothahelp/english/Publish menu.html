Publish menu

Publish ...

Clicking the Publish menu item gives the following three tabs :
  • Contents : defines contents
  • Parameters : defines what actions publish buttons will launch
  • Publish : launches Print or Export actions

OpenGotha Site

Starting from OpenGotha V3.37, Publishing on the internet can be done automatically. The opengotha.info site is dedicated to this.
To set up this feature,
  • Make sure your computer is connected to the Internet
  • In the Publish .. Parameters tab, select the Export to OpenGotha Site checkbox

You have nothing more to do. Each publish action from the Publish .. Publish tab will upload the corresponding file.

Manage OpenGotha Site for your tournament

In addition to the normal publish actions, you can :
  • Publish all possible files : Experimental tools .. Export all exportable html files (local and remote files)
  • Deleted all files : Experimental tools .. Delete all exorted files (remote files only)
Your files will be kept on OpenGotha site during at least one month. You may at any time update or delete your files.
